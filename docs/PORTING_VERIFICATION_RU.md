# Проверка переноса MT4 -> MT5

Документ фиксирует, что проверено по исходникам и по документам:

- `Инструкция советник.docx`
- `Техзадание на советник на МТ4, МТ5.docx`
- исходный код `MQL4/Experts/Acteck/Acteck.mq4`

## Что перенесено

1. Исторический анализ трендов по фильтрам волатильности.
2. Определение направления/смены направления:
   - `GetDirection`
   - `CheckUp`
   - `CheckDn`
3. Расчёт вероятностей по массиву пройденных участков.
4. Разметка на графике:
   - трендовые линии,
   - вероятностные уровни,
   - текущая и целевая метка.
5. Табличное представление инструментов и фильтров.
6. Переход на символ/ТФ по клику в ячейке таблицы.
7. Формирование отчётов в CSV.

## Где теперь лежит файл символов

MT5 путь:

- `MQL5/Files/Acteck/forex.set`

В коде используется:

- `Acteck\\<NameSet>.set`

Это полностью соответствует подходу из MT4-версии, где использовался файл в `MQL4/Files/Acteck/`.

## Что изменено технически

1. В MT5 используется нативный UI на графических объектах (без MT4 GUI framework).
2. Вызовы рыночных данных адаптированы под MT5 API:
   - `MarketInfo` -> `SymbolInfoDouble/SymbolInfoInteger`
   - `iATR` через handle + `CopyBuffer`.
3. Авторские данные в файле обновлены на:
   - `Copyright: Evgeniy Acteck`

## Ограничения

- Для финальной проверки «1 в 1» нужно прогнать одинаковые участки истории в MT4 и MT5 и сверить:
  - итоговые CSV;
  - ключевые значения в таблице;
  - поведение при кликах и обновлении.
